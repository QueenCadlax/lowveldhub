## 🎨 DESIGN SYSTEM

### Color Palette
- **Primary**: Gold (#D4AF37, #C9A24D) — Luxury, CTAs, accents
- **Background**: Black (#050505), Dark Slate (#0F172A)
- **Cards**: White/10% opacity glass effect
- **Borders**: White/10%-15% opacity
- **Status Colors**:
  - Essential: Slate-600
  - Professional: Blue-600
  - Platinum: Purple-600 → Indigo-600

### Typography
- **Headings**: Serif font (luxury, premium feel)
- **Body**: Sans-serif (clean, readable)
- **CTA**: Bold, uppercase tracking
- **Secondary**: Light font weight, opacity 70%

### Spacing & Layout
- **Container**: Max-width 7xl with padding
- **Cards**: 24px padding, 8px border-radius
- **Grid**: 1 col mobile → 2 cols tablet → 3-4 cols desktop
- **Gap**: 16px between items

### Animations
- **Hover**: Subtle lift (transform: translateY(-4px))
- **Focus**: Border + glow effect
- **Loading**: Spinner rotation
- **Transitions**: 300-400ms ease timing
