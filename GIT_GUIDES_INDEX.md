**What:** PowerShell script to automate everything  
**Use:** PowerShell terminal  
**Time:** 2-3 min to run  
**Best for:** Most users (colorful output, error handling)


**What:** Batch script (no setup needed)  
**Use:** Command Prompt or double-click  
**Time:** 2-3 min to run  
**Best for:** Users who prefer traditional batch files

```bash
git-init.bat
```
